Solution: Composition
Answer: DREAMINESS

Written by Deusovi and Level 51

The puzzle has a lot of emoji... but also a lot of weird characters with dotted lines. Looking them up, you'll find that these are ideographic description characters. They're used with Chinese characters, as a way of describing how certain characters are made up of others. For example, ⿰口犬 = 吠, and ⿴囗人 = 囚.

You can nest these within each other, using prefix notation: ⿱⿰⼒⼝馬 represents the character 駕.

So, this puzzle asks you to do the same thing, but with emoji instead! If you go through the laborious process of constructing the very large square image (or more likely, gradually remove bits without assembling the full thing), you'll notice the overall structure: it consists of 11 nested images.

The outer 10 images all represent paintings:
