Solution: Harmony
Answer: SWISHED

Written by moo

Upon opening the puzzle, solvers are greeted with seven chess positions, with dashes along the side of each chessboard. The flavor-text (specifically "Queen" and "gambits") hints at the popular Netflix show The Queen's Gambit. Indeed, the star of that show is named Beth Harmon, and the puzzle title references that.

The flavor-text again emphasizes that we need to find the Queen's next move. To do so, we will have to find the games from which each of the positions occurred in The Queen's Gambit. The best article for doing so is here, which should show up relatively quickly as solvers search for it. The opponent's names fit in the blanks on the side of each position, written from top to bottom; by using the row that the next move is played on, we can extract a letter from each position, as follows (note, this table uses algebraic notation):

Opponent's Name Next move played Extracted Letter
Beltik Bf5 I
Diedrich Rac8 H
Duhamel Qd7 D

Shaibel Qxf7 S
Townes Nb3 W
Watts Nd3 S

The opponent's names are given in alphabetical order; rearranging the extracted letters by the order in which the games are played in the show gives the final answer SWISHED.
