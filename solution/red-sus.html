Solution: RED SUS
Answer: ENVIRONMENT

Written by lovemathboy and chaotic_iak

This puzzle contains 10 logic puzzles listed in alphabetical order. However, within each ruleset, some words or phrases are written in red instead. These phrases should be interpreted as giving rules that are strictly incorrect.

Fillomino (by chaotic_iak)

"If two regions are adjacent, they do not have the same size": If two regions are not adjacent, they do not have the same size. It's easier to think about the contrapositive of this rule, which is "If two regions have the same size, then they are adjacent." Hence, all regions of the same size are pairwise adjacent.

Extracted word: SENTIMENTAL

Hashiwokakero (by lovemathboy)

"If there exist bridges between 2 islands, there are either one or two bridges between them": If there's a bridge between islands, then there must be at least 3 bridges between the islands.

Extracted word: SMORGASBORD

Heteromino (by chaotic_iak)

"Regions of the same shape and the same orientation are not orthogonally adjacent.": Pretty self-explanatory ruleset, each triomino can only be adjacent to a triomino with a different shape or a triomino with the same shape and orientation.

Extracted word: ACCOUNTANCY

Icebarn (by lovemathboy)

"The directed path may cross only on ice cells.": The path is allowed to cross on non-ice cells and not allowed to cross on ice cells.

Extracted word: IRREVERENCE

LITS (by lovemathboy)

"Same tetrominoes may not be adjacent.": Different tetrominoes may not be adjacent. Another way to phrase this is if 2 tetrominoes are adjacent, then they are the same shape. "All shaded cells must form a connected region": Pretty self-explanatory, every non-shaded cell must form a connected region. "There may not be any 2x2 region of shaded cells": Also self-explanatory, no 2x2 region of non-shaded cells are allowed.

Extracted word: GRANDFATHER

Masyu (by lovemathboy)

"Draw a single non-intersecting loop connecting centers of cells via axis-parallel lines.": The loop must still go from center to center, but each loop segment cannot be axis-parallel.

Extracted word: CONSPICUOUS

Moon or Sun (by lovemathboy)

"A region of bolded lines is called a "room". The loop goes through each room exactly once.": In combination with another rule, this implies the loop must visit each room at least once. In rooms with both moons and suns, this rule implies one visit must visit all suns and no moons while another visit must visit all moons and no suns. However, if a room lacks a type of clue, the loop can make multiple "visit all shapes of that type" visits to that room.

Extracted word: MANUFACTURE

Nurikabe (by lovemathboy)

"Each unshaded region contains exactly one number.": Each region must contain either 0 or at least 2 numbers. Since a number depicts the size of the region it's in, all numbers within the region must be the same.

Extracted word: OXYGENATION

Star Battle (by lovemathboy)

"Each row and column contains exactly 2 stars": Each row and column can contain any number of stars except for 2.

Extracted word: DIMENSIONAL

Sudoku (by lovemathboy)

"Every row and column contains each number 1-9 exactly once.": Each row and column must contain each number 1-9 at least twice or none at all. Since each box contains each number from 1-9 exactly once, this implies 3 rows with the boxes must contain each number from 1-9 exactly 3 times. If a number appears exactly twice in a row, there will be no other way to pair up the last remaining occurrence of that number. Hence, this implies each row and column must either contain exactly 3 of a number or none of that number.

Extracted word: PRECIPITATE

Extraction

Author’s Notes

lovemathboy: I love writing logic puzzles with cursed rulesets... so this puzzle was born. As a fun fact, Moon or Sun was so cursed that I interpreted my own ruleset wrongly...

Also, it can't be a lmb puzzle without an alignment chart at the end :)

chaotic_iak: lovemathboy asked me to help write some cursed logic puzzles, so I did. After thinking about my favorite genres, I thought of Fillomino and this rule change, then I realized it could actually make for an interesting puzzle, so I decided to tackle it. Later on, lovemathboy already came up with nine other puzzles, but one of them (a Slitherlink) was rather weak (read: not cursed enough) and I saw a lack of region division puzzles, so I decided to write another, giving the Heteromino. Also, it would be strange if an author only writes one of the ten logic puzzles.
